进入控制台 → 参考文档
app.ts - OpenAI SDK
import OpenAI from "openai";

// 替换为 Arqel 的入口与凭证，代码无需其他修改
const client = new OpenAI({
  baseURL: "https://api.arqel.net/v1",
  apiKey: "arq-sk-xxxxxxxxxxxxxxxx",
});

const response = await client.chat.completions.create({
  model: "claude-3-7-sonnet-20250219", // 自由切换任意模型
  messages: [{ role: "user", content: "Hello, Arqel!" }],
});

完善的排查工具

工具层出错不一定是模型挂了。Arqel 提供完整面板帮你快速定位问题所在。

独立凭证管理

可按模型服务商或 Agent 场景分别配置和撤销 Key，无需维护单一高风险全局凭证。

完整请求日志

保留每一次调用的 Request Payload 与响应状态码，工具调用报错时一查便知。

分组路由规则

支持将不同的代码补全、审查等任务自动分配到价格与速度最匹配的底层模型。

用量消耗透明

实时统计各模型 Token 消耗及账单费用，支持按项目、按日期维度导出核对。

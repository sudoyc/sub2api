把「下一步怎么做」放到页面里。

首页不需要写长篇教程，但应该给用户一个可复制、可理解的配置轮廓。

# 1. 在 Arqel 创建 API Key
ARQEL_API_KEY="sk-..."

# 2. 在你的 Agent / CLI 里配置同一个入口
BASE_URL="https://api.arqel.net/v1"

# 3. 按客户端选择模型
MODEL="claude-sonnet-4-5"      # Claude Code
MODEL="gemini-3-pro-preview"   # Gemini CLI
MODEL="gpt-5.4"                # Codex / OpenAI-compatible

先创建 Key

进入控制台创建 API Key，按需要分配分组、额度和有效期。

再选择 Agent

按 Claude Code、Gemini CLI、Codex CLI、OpenCode 等教程复制对应配置。

最后看用量

通过用量记录确认模型、Token、费用和错误原因，避免盲猜。

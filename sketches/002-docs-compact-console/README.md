## Variant: Compact console

### Design stance
偏工具台/命令面板，用步骤卡直接承载 docs 的核心动作。

### Key choices
- Dark/purple arqel baseline (#09090b / #6467f2)
- Thin borders, restrained cards, terminal/code motifs
- No Auto Route marketing claim
- Self-contained throwaway HTML

### Trade-offs
- Strong at: 效率高、很 docs；适合技术用户。
- Weak at: 品牌首页感弱，视觉更紧凑。

### Best for
- Choosing a docs homepage direction before implementing in VitePress.

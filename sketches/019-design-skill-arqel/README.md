# Arqel sketch 019: design-skill test

基于 `sketches/gemini.html` 的三个 disposable HTML 方向。

## 01 Linear Command Split
- 设计立场：保留深色紫色 arqel 基调，把右侧终端升级为模型路由命令面板。
- 强项：开发者工具感强，右侧 tabs 可交互，适合强调控制台能力。
- 弱项：营销情绪较冷。

## 02 Vercel Light Precision
- 设计立场：用 Vercel 式 shadow-border、大片留白和浅色精确感测试同语言的 light theme。
- 强项：干净、可读、生产感强；有明暗切换。
- 弱项：相比当前 gemini 深色方向，品牌记忆点更弱。

## 03 Scroll Narrative Console
- 设计立场：把首页变成解释型叙事，右侧 sticky 状态卡持续提供产品上下文。
- 强项：最适合讲清楚接入/责任/消耗边界。
- 弱项：首屏冲击力不如 01，页面更偏说明文档。

# 模型选择

模型名决定这次请求要调用哪个具体模型。常见模型家族包括 Claude、GPT、Gemini 等，但实际填写时需要使用 Arqel 控制台中展示的可用模型名。

## 直接指定模型

调用时需要明确指定模型：

```json
{
  "model": "请替换为控制台里可用的模型名"
}
```

适合：

- 你明确知道要使用哪个模型。
- 你的提示词或输出格式依赖某个模型特性。
- 你需要稳定复现行为。

## 从哪里找到模型名

建议从 Arqel 控制台复制模型名，不要凭经验手写。

常见错误包括：

- 把模型显示名称当成模型名。
- 多写或少写版本后缀。
- 使用了当前账户不可用的模型。
- 在工具中填写了另一个平台的模型名。

## 如何选择模型

- 写代码、改项目：优先选择你熟悉且上下文能力足够的模型。
- 简短问答、翻译、摘要：可以选择成本更低、响应更快的模型。
- 需要稳定输出格式：固定使用同一个模型，不要频繁切换。
- 不确定时：先选一个控制台推荐的通用模型，跑通后再比较效果和消耗。

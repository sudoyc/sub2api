# 接入检查清单

接入前确认：

- 已按系统完成环境准备：macOS、Windows 或 Linux / WSL。
- `node -v`、`npm -v`、`git --version` 正常返回。
- 已创建 API Key。
- 已复制 Base URL。
- 已确认要使用的模型名。
- 已确认工具支持自定义 API Key 和 Base URL。
- 已准备一个简单问题用于测试。

接入后确认：

- 第一条请求返回成功。
- 控制台能看到用量记录。
- 工具里的错误提示为空。
- 已确认当前工具里填写的是一个 Arqel 支持的具体模型名。

上线前确认：

- 生产环境和测试环境使用不同 Key。
- Key 没有写入前端代码。
- 异常状态有日志或提示。
- 已设置必要的额度或用量提醒。

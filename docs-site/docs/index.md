## 推荐阅读路径

1. 先读 [从 0 开始](/getting-started/)，理解 Arqel 解决什么问题。
2. 按你的系统完成 [环境准备](/setup/)：macOS、Windows、Linux / WSL 分开处理。
3. 按 [创建 API Key](/getting-started/api-key) 和 [发送第一条请求](/getting-started/first-request) 跑通请求。
4. 根据你的工具选择 [Cursor](/tools/cursor)、[Claude Code](/tools/claude-code)、[Gemini CLI](/tools/gemini-cli) 或 [Codex CLI](/tools/codex-cli)。
5. 遇到问题时查看 [请求失败排查](/help/troubleshooting)。
